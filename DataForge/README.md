# DataForge 🛠️

Premium AI-powered data quality & standardization engine for data teams.

Automates repetitive data cleaning—parsing messy files, normalizing values, 
detecting anomalies—while maintaining full control and transparency.

**Features:**
- 📊 Schema profiling with semantic type inference
- 🚨 Anomaly scanning (outliers, rare categories, format deviations)
- 🧠 AI-assisted operations (LLM column inference & value normalization)
- 🔀 Fuzzy deduplication using Levenshtein distance
- 🛡️ Custom validation rules & quality assessment
- 💾 Rich exports (cleaned data + quality reports)

**Tech:** Python · Streamlit · Pandas · RapidFuzz · Claude & Gemini APIs · Pytest · Docker

**Built for:** Data engineers & analysts at fashion brands, retailers, healthcare systems, 
manufacturers, and financial institutions.
